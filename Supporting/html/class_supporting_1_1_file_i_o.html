To close the database file by writing an array of strings to the database file.

Details

This method will take in an array of strings and write them to the database file that is specified as the second parameter. The number of records written as well as the number of valid and invalid records written will be logged.

Parameters
dbName- string - the full pathname of the database file
stringsToWrite- string[] - the string array that contains the strings to write to the file
Returns
Nothing is returned
static string [] Supporting.FileIO.OpenDBase ( string  dbName)
static

To read the database file and validate the records.

Details

This method will open a StreamReader for the dbReader data member and call readRecord() to read each line from a database file. Each line will be stored in as a separate string and once the read is done each string will be checked if it is valid and the number of reads as well as valid and invalid records will be logged. The string array of valid records will then be returned.

Parameters
dbName- string - the full pathname of the database file
Returns
An array of strings validRecords that will hold all of the valid records read in the database file
